How To Apply

  • 1. Make sure to close the game if it is currently running.
  • 2. Extract the update file and run "OshiRabu - append updater.exe".The installation will start. Please follow the instructions.
  • 3. Select the installation folder for "OshiRabu: Waifus Over Husbandos".


*To check where your game is installed, go to your Steam library and select
"OshiRabu: Waifus Over Husbandos".
From the menu, select "Manage" then "Browse Local Files".
The folder indicated is the folder where you should install the patch.
[ ～ Steam\steamapps\common\OshiRabu Waifus Over Husbandos Demo ]
If you choose a folder where the game is not already installed, an error will occur and the installer will close.

Regarding Save Data

Your save data should be compatible, but if there is a problem with the installation you may need to delete your save data.Your save data is stored in the UserData folder found within the "Oshirabu: Waifus Over Husbandos" installation folder.
